## Run Katas with [Jasmine](http://jasmine.github.io/)

Source code contains

      package.json
      bower.json
      gulpfile.js

Simple steps to build

      // single-time install
      npm install
      npm install -g bower
      npm install gulp
      npm install bower

      // run gulp tasks
      gulp vendor
      gulp build

After this steps simply open index.html to view test results produced by karma boot.js
